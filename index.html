Project 1

A custom email, with landing page and Google Maps custom routes

Project executed for Cadogan Tate Ltd.

The Breif

The original brief I was given is as follows, the email and landing page must;

  • Feature an interactive map.
  • Provide information about the delivery service Cadogan Tate offer, namely the different routes and their respective departure dates.
  • Work on both mobile and desktop.
  • Be similar in design to the new website.
  • Work on all major email clients.
  • Provide a simple and seamless user experience.
  • Push customers to enquire about the “US shuttle service” being offered.

The initial plan & finalising a design

The weekly “US Shuttle Service” email has been a successful part of Cadogan Tate’s marketing strategy for the past 4 years, but an updated design was well overdue. Originally a co-worker was managing this project but making little progress with the client on finalising a design, so I was put in charge to execute this project promptly.

I decided to start fresh in order to ensure I didn’t miss any important elements of this project and scrapped the previous design that hadn’t met the client's requirements. The design of the email was going to be based off the client's current website, so firstly I put together a brand resources file containing their brand colours, fonts, logos and general design features of the website. I used this file as inspiration when designing the email.
